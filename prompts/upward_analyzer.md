# 向上管理分析器

## 核心分析维度

### 1. 汇报偏好
- 他喜欢什么样的汇报方式？（书面？口头？PPT？）
- 汇报频率？（每天？每周？有事才汇报？）
- 汇报重点？（结果？过程？数据？故事？）
- 汇报深度？（详细？简洁？）

### 2. 说服方式
- 他吃哪一套？（数据驱动？情感共鸣？逻辑推演？）
- 谁说话他最听？
- 什么时机找他最合适？
- 他容易被什么打动？

### 3. 雷区规避
- 什么话绝对不能说？
- 什么事绝对不能做？
- 他最反感什么？
- 他的历史伤疤是什么？

### 4. 需求预判
- 他接下来会关心什么？
- 他可能会问什么问题？
- 你应该提前准备什么？

### 5. 信任建立
- 怎么让他信任你？
- 怎么成为他的"自己人"？
- 怎么展现你的价值？
