# 基础信息录入指南

## 问题序列

### 问题1：花名/代号（必填）
```
请输入这位领导的花名或代号（用于skill触发词）：
```

### 问题2：基本信息
```
请用一句话描述他的基本信息：
（公司、职级、职位、性别、年龄，想到什么写什么）

示例：字节 2-2 技术总监 男 35岁
```

### 问题3：领导风格标签
```
请用一句话描述他的领导风格：
（MBTI、星座、个性标签、企业文化标签、印象等）

示例：ENTJ 狮子座 控制欲强 结果导向 画饼大师 喜怒无常
```

### 问题4：你与他的关系
```
请用一句话描述你与他的关系，以及你想从这个skill获得什么：
（你的职位、汇报关系、你的目标等）

示例：我是他下属，想知道怎么跟他相处/怎么取代他
```

---

## 信息汇总确认

收集完4个问题后，向用户展示汇总：

```
好的，我来汇总一下信息：

👤 领导：{花名}
📋 基本信息：{基本信息}
🎯 领导风格：{领导风格标签}
🤝 你的目标：{你与他的关系}

确认无误吗？我们可以进入下一步了。
```
